**时间**: 2024.11 -- 2024.12

### 项目概述
本项目旨在打造一个能够理解自然语言指令并协同视觉感知完成物理操作的“机器脑”系统，探索大语言模型在机器人控制领域的应用潜力。

### 核心技术
1.  **大模型本地化与优化**: 
    *   负责 LLM 的本地化部署，通过量化、边缘化部署与并行化技术，结合 Prompt 工程优化。
    *   成功将首词生成延迟从 **4.0s 降低至 1.4s**，显著提升了人机交互的流畅性和实时性。
2.  **闭环执行 Pipeline**: 
    *   构建了完整的“语言-视觉-动作”链路：用户指令 -> LLM 解析 -> **Grounding DINO** 开放词汇物体定位 -> 点云数据融合 -> **AnyGrasp** 生成抓取位姿 -> 机械臂执行。
3.  **场景验证**: 
    *   在桌面物品整理、汉诺塔求解等复杂任务中进行验证，整体操作成功率超过 **90\%**。
    *   系统支撑了多次校内外重要参观演示，获得高度评价。
